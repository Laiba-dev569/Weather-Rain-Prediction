# 🌧️ Rainfall Prediction using Machine Learning

## 📌 Overview
This project predicts whether it will rain tomorrow using the Australian Weather dataset. It demonstrates a complete machine learning workflow, including data cleaning, preprocessing, feature engineering, exploratory data analysis (EDA), model training, and evaluation.

## 📂 Dataset
- weatherAUS.csv

## 🛠️ Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Workflow
- Data Cleaning
- Data Transformation
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation

## 🤖 Machine Learning
- Random Forest Classifier

## 📈 Future Improvements
